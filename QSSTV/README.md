## Custom QSSTV Script for Linux
### [:arrow_down: Download](qsstv.sh)

## Installation
* Step 1:   `apt-get install qsstv`
* Step 2:   Download the Script
* Step 3:   `chmod +x qsstv.sh`

## Usage
* **Run:**    `./qsstv.sh <audio-filename>`
* **Stop:**   `[Ctrl + c] to stop playback`
